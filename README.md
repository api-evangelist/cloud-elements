# Cloud Elements (cloud-elements)
Cloud Elements was a Denver-based API integration platform that offered pre-built connectors ("Elements") and category-level Unified APIs (CRM, Documents, Finance, Helpdesk, Messaging, etc.) so SaaS vendors could embed third-party integrations without building each connector from scratch. UiPath acquired Cloud Elements in March 2021 and has since folded the technology into UiPath Integration Service. This profile is retained as a historical record and a pointer to the successor platform.
